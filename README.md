📖 About the Project

This project is a static HTML-based online bookstore that showcases and sells educational and knowledge-based books in multiple Indian languages.

It focuses on:

- Accessibility (♿ inclusive design)
- Multi-language support
- Simple and direct payment integration
- Clean and responsive UI

- Telugu (తెలుగు)

- Hindi
